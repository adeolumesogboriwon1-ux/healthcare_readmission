# Healthcare Patient Readmission Analysis

## Overview
Predict early hospital readmission for diabetic patients.

## Dataset
- 100,244 patient records
- Source: UCI Diabetes 130-US Hospitals

## Results
- Model: Random Forest Classifier
- Overall Accuracy: 89%
- Early Readmission Rate: 11.22%

## Key Findings
1. Age group 20-30 has highest readmission rate (14%)
2. Longer hospital stays increase readmission risk
3. Early identification can reduce hospital costs

## Tools Used
Python, pandas, scikit-learn, matplotlib, seaborn
